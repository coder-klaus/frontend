## 数据库 DDL

```sql
-- 查看系统所有数据库
SHOW DATABASES;

-- 查看当前数据库
SELECT DATABASE();

-- 创建数据库
CREATE DATABASE `music_db`; -- 无论数据库是否存在，都会创建，但如果数据库存在，会报错
CREATE DATABASE IF NOT EXISTS `music_db`; -- 只有当数据库不存在时，才会创建

-- 使用某个数据库
USE `music_db`;

-- 删除数据库
DROP DATABASE `music_db`; -- 无论数据库是否存在，都会删除，但如果数据库不存在，会报错
DROP DATABASE IF EXISTS `music_db`; -- 只有当数据库存在时，才会删除
```



### 内置数据库

```sql
mysql> SHOW DATABASES; # 查看所有数据库
+--------------------+
| Database           |
+--------------------+
| information_schema | # 记录了MySQL 有哪些数据库，有哪些数据表，对应有什么字段，所需要的访问权限是什么
| mysql              | # 记录了MySQL 有哪些用户，有哪些权限
| performance_schema | # 记录了MySQL 的性能信息
| sys                | # information_schema 和 performance_schema 的汇总, 以更精炼的格式和图表来展示结果
+--------------------+
6 rows in set (0.099 sec)
```
